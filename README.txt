Погодное приложение, которое отображает погоду в определенном городе.

В ходе создания сайта был использован Framework Django.

В ходе выполнения проекта, был использован Bootstrap, для создания "шапки" главной страницы сайта.

После этого создается основной блок. Внутри этого блака расположено еще два блока, которые расположены рядом.
Внутри первого блока расположена текстовая форма с кнопкой (при нажатии можно узнать погоду в вашем городе).
В другом блоке расположена информация в другий городах. Дизайн можно создать другой.

Для синхронизации погоды необходимо воспользоваться уже готовым погодным сайтом. Я выбрал сайт
OpenWeatherMap. Нужно взять на этом сайте API ключ(файл views.py строка №8). Для того чтобы отправить API запрос,
нужно взять url адрес(файл views.py строка №9).

Для отправки url запроса и получения его обратно, был установлен модуль requests.

В файле models.py была создана таблица(City) для всех городов.
В файл admin.py была добавлена таблица из файла models.py для отображения ее в понели администратора.
После чего в самой понели администратора добавляем новые значения(названия городов).
Все эти значения добавляются в таблицу и выводятся в блоке №2 с помощью цикла. Данный цикл записан в
файле views.py.

Далее добавляется функционал. Создается файл forms.py. В данном файле импортирует класс ModelForm
для того чтобы форма была похожа на понель администратора. И дополнительно импортируется класс
TextInput. Он позволяет добавить атрибуты к полям формы.
